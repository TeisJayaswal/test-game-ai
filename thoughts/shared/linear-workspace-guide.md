## MCP Tool Usage Examples

### Create Issue
```
mcp__linear__create_issue
- title: "Issue title"
- team: "Normcore" (default team)
- assignee: "max" (display name, email, or ID)
- priority: 2 (High)
- labels: ["Feature", "Bug"]
- description: "Markdown description"
- project: "Claude Code Agent" (name or ID)
- state: "Todo" (status name or ID)
```

### Update Issue
```
mcp__linear__update_issue
- id: "issue-id" (required)
- state: "In Progress"
- assignee: "teis"
- priority: 1
```

### List Issues
```
mcp__linear__list_issues
- team: "Normcore"
- assignee: "me" (for current user's issues)
- state: "In Progress"
- project: "Claude Code Agent"
```

### Blocking Relations
When creating issues, you can set blocking relationships:
- `blocks`: Array of issue IDs this issue blocks
- `blockedBy`: Array of issue IDs that block this issue
